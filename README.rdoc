= GaXy
Космическая леталка

==Технические данные
* игра использует движок Unity3D 5.3.1 PersonalEdition
* язык C#

==Об игре
* цель игры перевозить ресурсы с планеты на планету
* во время перелета необходимо уничтожать врагов

==Основы и Цель проекта
* развлекательно-позновательный
* доступность
* принципиальное отсутствие коммерции в каком-либо виде
